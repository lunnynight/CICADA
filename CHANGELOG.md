# Changelog

## [0.1.0] - 2026-03-10

### Added
- Dashboard with real-time OpenClaw service status polling
- Setup wizard with automated Node.js (winget) and OpenClaw (npm) installation
- Model configuration page supporting 7 domestic + 3 international providers
- API connection testing for all provider types
- Ollama local model auto-detection
- Config persistence to `~/.openclaw/openclaw.json`
- Skills store with search and one-click install/uninstall
- OTA update checker via GitHub Releases API
- Terminal output widget with color-coded log display
- NPM mirror selection (Taobao, Tencent, Huawei)
- Dark theme UI with Material 3
- Windows x64 release build (29MB)
- GitHub Actions CI/CD for automated releases
